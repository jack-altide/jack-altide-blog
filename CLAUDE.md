# enPiT Blog Project

## プロジェクト概要
- Astro製の静的ブログサイト（enPiT個人レポート用）

## enPiT 個人レポート要件

### 必須要件
- enPiTでの取り組みをブログ記事形式でまとめる
- 以下の **4つの観点** を必ず含める（順序や中身はアレンジOK）
- **最終のチームAMFは必ず含めること！**

### 4つの必須観点

1. **開発したプロダクトについて**
   - プロダクトの概要、使い方など

2. **チームについて**
   - メンバー構成、それぞれの役回りなど

3. **秋学期中のプロダクト・チームの軌跡**
   - 各スプリントでやったこと、起きたこと、学んだことなど

4. **期間を通した個人的ふりかえり**
   - 学んだこと、それを次に何に活かすかなど

### 参考：昔のレポートフォーマット（含めなくてもOK、内容考える際の参考）

- 開発したシステムの内容
  - 最終的なリーンキャンバスを各個人で整理
  - （顧客、問題、独自の価値提案、ソリューションは必須）
- 開発したシステムの使い方概要
- 期間中の自分の主な役割
  - 明確な実装担当がない場合も、チームの中での役回りを書こう
- 秋学期の各スプリントの個人的ふりかえり
  - 各スプリントについて（チーム&自分）やったこと、起きたこと、学んだこと
- チームメンバーについて
  - 各メンバーのチームでの役割りと貢献を個人的解釈で
- 期間通した個人的振り返り
  - 学んだこととと、それを次に何に生かすか
